# 立即备份

当用户说"立即备份"或类似词语时，执行系统备份命令。

## 触发条件

用户输入包含以下任一关键词：
- "立即备份"
- "马上备份"  
- "现在备份"
- "立刻备份"
- "执行备份"
- "手动备份"

## 执行动作

1. 执行系统命令：`python3 /home/node/app/backup_controller.py manual`
2. 该命令会**强制备份**，不受自动备份的5分钟间隔限制
3. 等待命令完成并获取输出
4. 向用户报告结果

## 输出格式

- 成功时回复："✅ 立即备份已完成，数据已保存到 Hugging Face Dataset"
- 失败时回复："❌ 备份失败，错误信息：{stderr}"
- 如果正在备份中："⏳ 另一个备份正在进行中，请稍后再试"

## 注意事项

- 该命令会**立即执行**，即使5分钟内已经备份过也会再次备份
- 自动备份的5分钟间隔保护仍然生效（对自动变化检测和定时备份）
- 备份范围包括：sessions、workspace、agents、skills
